File Folder Description

CONUSUrbanForestThreatsto2060.csv \Data Comma-separated values (CSV) file containing numerous threats to urban forests assessed for the conterminous United States such as projected changes in urban tree cover, air temperatures, precipitation, aridity, sea level rise, wildfires, and flooding, as well as threats from hurricanes, tornadoes, ice storms, and insects and diseases. Also included is a cumulative threat index.
